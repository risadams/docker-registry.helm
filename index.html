docker-registry Helm chart

A maintained fork of twuni/docker-registry.helm.

This is the Helm chart repository. Add it with:

helm repo add docker-registry https://risadams.github.io/docker-registry.helm
helm repo update
helm install my-registry docker-registry/docker-registry

Or pull the chart from the GHCR OCI registry:

helm install my-registry oci://ghcr.io/risadams/docker-registry

Source & documentation: github.com/risadams/docker-registry.helm
