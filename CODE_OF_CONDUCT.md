# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code of
conduct. The full text is maintained by the Contributor Covenant project:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

By participating in this project — filing issues, opening pull requests, or
otherwise engaging in the community — you agree to abide by its terms. In short:
be respectful, welcoming, and constructive.

## Reporting

To report unacceptable behavior, contact the maintainer privately through their
[GitHub profile](https://github.com/risadams). Reports will be reviewed and
handled confidentially.

Security vulnerabilities are handled separately — see [SECURITY.md](SECURITY.md).
